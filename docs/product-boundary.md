# 产品合规边界

本项目定位为**个人笔记美化、扫描稿恢复、模板重排版、个人手写风格排版**工具。
所有功能仅用于处理**用户本人拥有的扫描稿**和**用户本人的手写风格**。

## 1. 允许的使用场景

- 个人课堂/读书笔记的排版美化
- 个人手写草稿的电子化整理与重排
- 用户本人旧扫描稿的清理与修复（去污、去字迹残留、补背景）
- 用户本人手写风格的文字排版与版式调整
- 个人模板的批量排版

## 2. 禁止的使用场景

下列功能**不做**、**不提供**、**不在文案中暗示**：

| 类别 | 说明 |
|------|------|
| 签名 | 不实现任何签名生成、签名模仿、签名替换 |
| 证件 | 不处理身份证、护照、驾照等证件图像 |
| 合同 | 不用于合同、协议、公文等具有法律效力的文件篡改 |
| 票据 | 不处理发票、收据、银行回单等票据 |
| 考试/作业冒充提交 | 不用于冒充他人提交作业、考试答卷 |
| 伪造他人笔迹 | 不采集、不模仿、不生成他人的笔迹风格 |
| 规避鉴定 | 不针对笔迹鉴定、文件真伪检测做任何对抗处理 |

## 3. 文案规范

在 UI、文档、营销材料中：

- **不使用**「伪造」「骗过检测」「规避审核」「以假乱真」「冒充」等表述
- 描述功能时使用「美化」「整理」「重排」「恢复」「清理」等中性词汇
- 涉及去字迹功能时，说明用途为「清理个人草稿残留」「去除扫描噪点」

## 4. 技术边界约束

- `services/vision` 的去字迹能力仅接受**用户手动框选**的区域，不提供自动整页字迹识别与擦除
- 不内置任何他人笔迹模板、字体包
- 不提供「一键模仿某人的笔迹」入口
- 导出图像不嵌入任何用于溯源对抗或鉴定规避的隐藏标记

## 5. 责任声明

用户须保证所处理图像为本人所有或已获得合法授权。
因用户将本工具用于禁止场景而产生的法律责任，由用户自行承担。

## 6. 手写样本采集的合规约束

第三轮引入的个人手写样本库功能，仅用于采集**用户本人**的书写风格：

- **只采集本人笔迹**：用户只能导入自己手写的样本图，不得采集、上传、模仿他人的笔迹（包括但不限于他人签名、批注、答卷、手稿）。
- **用途限定**：采集的字形仅用于个人笔记排版、模板美化、扫描稿恢复，不得用于冒充他人提交作业 / 考试 / 签名 / 公文。
- **文案规范**：描述该功能时使用「采集你的个人书写风格，用于笔记排版和模板美化」，不使用「伪造」「冒充」「模仿他人」等表述。
- **无对抗检测设计**：字形渲染不做任何针对笔迹鉴定、真伪检测的对抗处理。
- **本地存储**：手写样本与字形数据存储在用户本地（项目 JSON / localStorage），不上传任何服务器。
