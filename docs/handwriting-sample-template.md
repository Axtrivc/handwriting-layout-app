# 手写样本采集模板

> 采集你**本人**的书写风格，用于个人笔记排版和模板美化。
> 请勿采集或上传他人的笔迹。

## 为什么要采集样本？

应用会把你的字形切割成小图，排版时优先使用你**真实的字形**，让笔记看起来像你自己写的，而不是印刷体。

## 采集规范

1. 用白纸、黑笔（中性笔 / 钢笔均可），光线均匀
2. 字写得**大一点、稀疏一点**，字与字之间留空隙，方便框选切割
3. 拍照或扫描时保持平整、俯拍、对焦清晰
4. 一行写一类内容，每类多写几遍（同一字符多 variant 效果更好）

## 如何导入

1. 在应用左侧打开「手写档案」面板
2. 新建档案（如「我的笔迹」）
3. 「导入样本图」→ 选择拍好的样本照片
4. 进入「切割字形」：
   - **自动模式（推荐）**：点「自动检测字形区域」，系统自动框出候选字框，再用「批量标注」一次性输入一行字
   - **手动模式**：在样本图上框选一个字 → 输入对应字符 → 点「保存字形」
   - 同一个字可多次框选，保存多个变体
5. 回到画布，文本对象切换渲染模式为「手写素材」，即可使用你的字形

## 如何让自动切字更准确

自动检测基于字迹的连通区域分析（不是 OCR，不识别具体字符），以下做法能显著提升候选框质量：

- **保持适当间距**：字与字之间留出明显空隙，避免笔画相连
- **横向排列**：一行字从左到右写整齐，系统按行聚类 + 阅读顺序排序
- **深色笔 + 浅色纸**：黑/蓝笔 + 白纸对比度高，二值化更干净
- **大小均匀**：字写得差不多大，过小（<8px）会被当噪点过滤
- **避免倾斜**：尽量横平竖直，倾斜会导致行聚类错乱
- **光线均匀**：避免阴影、反光，拍照时俯拍、平整
- **合并粘连**：如果两个字连在一起，可在候选框里手动拖拽调整或删除多余框

自动检测后仍可手动调整：删除错误框、拖拽移动、四角缩放、新增框，再批量标注字符。
